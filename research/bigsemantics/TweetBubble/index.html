TweetBubble is a free and open source Chrome extension that helps Twitter users explore social media associations. The extension makes it possible to see related content, user profiles, and tweets in-context, without switching back and forth between tabs and windows. By maintaining context, TweetBubble enables users to see the big picture and keep on track.

TweetBubble utilizes principles of interaction design and cognitive psychology to make the browsing a more fun and fluid experience. We extend our technique of in-context expansion to facilitate in-situ encountering diverse information on Twitter. This helps user explore multiple perspectives on a topic.

TweetBubble is a project of graduate students Ajit Jain, Nic Lupfer, Yin Qu, and Rhema Linder, and their advisor, Andruid Kerne. We use Metadata In-Context Expander from the BigSemantics framework to visualize metadata, and Meta-Metadata language to specify the data models, extraction rules and presentation semantics. The presentation semantics define rendering guidelines, creating visual consistency with Twitter and reducing redundancy.


Download the Chrome extension from the Web Store. Check out the open source TweetBubble code on BigSemanticsJavascript repository on GitHub.
