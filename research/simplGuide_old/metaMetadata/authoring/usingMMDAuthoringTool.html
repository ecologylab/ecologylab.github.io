Starting

  1. Start Eclipse and run MMDRepositoryServer.java in MetaMetadataRespositoryService project.
  2. Visit any page in Chrome or FireFox (In which you have already installed Tool).
  3. You will have an "Activate" button in bottom right corner, click that to start Tool.
  4. MMD Creator, A movable and resiable Box will open.

UI interface

  1. name : The name of current MMD being authored.
  2. Selector : For url selector. Tool will automatically initialize it with page url (removing all get params).
  3. Xpath : Xpath of currently selected dom element or MMD Field Tag.
  4. Reset button : It restore Tool to its initial condition.
  5. Cancel button : Cancel and unselector current dom element.
  6. Load button : To load already authored MMD from repository. You can choose between extending and reusing.
  7. Test Xpath button : To test Xpath present in Xpath textbox. It will open up a box with all possible results.
  8. MMD Table : A ui implementation of authored MMD.

Testing Xpath

  1. Enter a Xapth in Xpath textbox.
  2. Click Test Xpath button. Preview section will show only first 5 results and Extracted Value Box will show all possible results.

Adding a Field Tag

  1. Right click on required dom element. An inspector section will assist you to navigate up, down, next, Prev and Add.
  2. Just click Add in inspector section or + button in MMD table. Xpath will be automatically generated and added.

Editing attributes of a Field Tag

  1. Left click on any attribute to make it editable in MMD Table e.g.

  2. Please note that composite/collection types will be available only when Field type is Composite or Collection else Scalar types will be avilable as String, int and ParsedURL.
  3. Please note presence of a green + button to add child/member when Field Tag type is Composite or Collection.

Adding Custom Attribute Field Tag

  1. click on + button in last column of MMD Field Tag.
  2. Click on + button to add custom attribute. Click on attribute name and value to edit. An auto suggest will help.
  3. Click ok to save.
  4. Please note change in color of + button in last column of MMD Field Tag to indicate presence of custom attributes.

Editing Custom Attribute Field Tag

  1. click on + button in last column of MMD Field Tag to open custom attributes box.
  2. Click on + button to add custom attribute.

Generate or Save MMD

  1. Click on genetare button to generate MMD in JSON format.
  2. Click save button (Currently this feature is not supported by Respository Service) to save MMS in repository.

Load or Extend already authored MMD

  1. Click on Load button. A box will open up.
  2. Enter url of authored MMD e.g. http://idle.slashdot.org/story/11/08/09/1356222/15-Billion-Star-Trek-Theme-Park-Coming-To-Jordan.
  3. Check Extend checkbox if you want to extend from authored MMD.
  4. Click Ok to load into Tool.
