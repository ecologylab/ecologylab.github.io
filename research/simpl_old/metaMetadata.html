We introduce meta-metadata: integrated representations that consistently describe how metadata can be extracted from information resources found in digital repositories and on the Internet, represented internally, acted on by software tools, and presented to users. Meta-metadata builds on s.im.pl serialization, using its descriptions of the structures of classes and fields to connect data structures with meta-meta-information, enabling development of reusable code for processing of information from templated web site sources by curators writing scripting language.
