In-Context Expander (ICE)

ICE is a JavaScript library for visualizing and interacting with meta-metadata type semantics. ICE allows developers to dynamically render semantics into rich interactive UI components using HTML and CSS. ICE relies on AJAX requests to a back-end service to provide the semantics and corresponding wrapper (which contain the needed presentation rules) for a given source document URL.
