Application Developer Tutorials

The set of tutorials is for Java application developers who wish to incorporate the meta-metadata framework. The tutorials will walk through setting up a development environment with all the proper projects and frameworks, reading from a meta-metadata repository, and building simple test applications including the implementations the application specific of the bridge functions.

The first tutorial, development environment setup, will cover downloading the needed Java projects and setting up a working environment. By the end of the first tutorial you will be able to run the example application provided.

The next tutorial, basic implementation, will go over in detail in specific implementations of the example application. The tutorial will go over the basics needed for you to create a simple Java application using meta-metadata.
