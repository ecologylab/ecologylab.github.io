  1. Buld JavaDocs. Look at the error messages that are generated. Resolve problems. Also add documentation as needed.
  2. Launch the news collection from the browser. After it's running fine for 5 minutes:
    • Save the composition space.
      • Close combinFormation. Restart the browser.
      • Launch with re:open.
      • Use File-Open to load the saved version of the composition space.
      • Turn the agent on, and continue to develop the composition.
      • Close combinFormation. Restart the browser.
      • Launch with re:open.
      • Use drag and drop from the file system to load the saved version of the composition space.
      • Close combinFormation.
      • Open the html of the saved composition. (In IE, you may have to select allow active content at the top of the window). Make sure that mouse over and navigation works. Click "launch in combinFormation," at the bottom of the page. Proceed to launch.
      • Turn the agent on, and continue to develop the composition.
  3. From the browser, launch a re:mix with 2-4 searches and websites that are of interest to you. Once it runs o.k. for some time:
    • Save the composition space without history.
    • Load the saved version of the composition space without history.
    • Save the composition space with history.
    • Load the saved version of the composition space with history.
  4. After launching from the web interface, try navigating to a container or container's hyperlink. Ensure that the page loads properly in the browser. Do this for both Internet Explorer and Firefox.
  5. Use drag and drop to bring elements in from a web page. Do this both with our extension installed in Firefox, and without it in Internet Explorer and Firefox. Save them to a file. Start a new session and open the file to ensure it opens properly.
  6. Create several new text elements in an empty space. Save the space. Restart combinFormation and restore the contents of the space to ensure that it opens properly.
  7. Edit some existing elements in a space. Save the space. Restart combinFormaiton and restore the contents of the space.
  8. Edit the metadata of some existing elements in the space. Save the composition, restart combinFormation, and reload the space. Ensure that the metadata displays and that the changes were preserved.
  9. Express interest in metadata terms using the in-context-slider. Save the composition, restart combinFormation, and reload the space. Ensure that the metadata term's interest levels have been preserved.
  10. Run any large search or collection for 8 hours, or until it crashes or becomes impossibly slow. In the console output, occasional debugging information will be shown indicating number of elements in the candidate pools and the free memory. If possible, notice these values at the time when the session stopped being productive, or when the 8 hours was up (whichever is first). The program should be able to run for at least the 8 hours.
