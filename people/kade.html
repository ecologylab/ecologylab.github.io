Kade Keith

I'm currently a front end developer at IBM. Specifically, I work on the UI for developers as get they started building applications on IBM Cloud. Additionally, I'm working part-time towards a MS in Computer Science (AI Track) from Stanford.

While an undergrad in the Interface Ecology Lab, I developed interfaces that saught to enhance the way people create, connect, discover, and interact with information on the web. I had two big projects while in the lab. The first was the IdeaMACHE Extension which does client side metadata extraction from web pages as a part of the BigSemantics architecture. The second was Monadic Metadata Explorer, which I created for my undergraduate research thesis. It took a monadic approach to network visualization to better help users navigate and comprehend large data sets. You can read the thesis here.
