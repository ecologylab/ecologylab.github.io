yin qu
Ph.D. Candidate, Computer Science
quyin cs.tamu.edu

I am a Ph.D. student working with Dr. Andruid Kerne at the Interface Ecology Lab. I am interested in semantic web and web semantics, exploratory browsing and search, human centered information retrieval, and application of machine learning approaches in supporting meaningful interactions.

My current projects include BigSemantics, a set of technologies that supports applications working with web semantics, and Metadata In-Context Expander (MICE), an experimental interface for transforming our experiences with the web.

BigSemantics features the meta-metadata language, with which developers can author wrappers for modeling, extracting and presenting web semantics. BigSemantics provides software libraries and a RESTful web service for extracting web semantics from regular web pages, without relying on websites to publish or maintain formal semantics. BigSemantics comes with a growing repository that addresses popular websites and semantic types. Check it out and build your own cool applications working with web semantics!

The MICE interface presents web semantics from different sources and webpages in one context, reducing the cognitive load of switching between browser windows and tabs. It helps alleviate the problems of disorientation and digression (losing track of whereabouts and goals) during web browsing. A new interface integrating different strategies of using the web, including browsing and search, is ongoing.

Before my Ph.D. study, I got my Bachelor's Degree from the College of Computer Science and Technology in Zhejiang University, at the beautiful city of Hangzhou in China, in 2009.
