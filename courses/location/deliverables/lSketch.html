location sketch
Use gps-based location tracking as the input modality for an interactive sketch. Use the ecologylabLocationAware library to drive an API/tool, either Google Earth or Worldwind. Manipulate graphics, text, and perhaps audio/video.
Explore. Engage the participant. Create a responsive environment. Think carefully about what data will be manipulated, how, and why. Create a mash-up that connects the GPS data with information from another source. Develop a concept and figure out how to implement it using one of these API/tools. Develop a sense of "response as a medium". Give the participant(s) an experience of place.
The goals are to learn to work with GPS and the API/tool you choose, and to develop a creative interactive idea.
Evaluation criteria:
  • Utilization of API/tool capabilities.
  • Interesting concept.
  • Interactive engagement of the human participant.
  • Affordance design.
  • Optional: social interaction.
